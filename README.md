# yomiage — 選択テキスト読み上げアプリ

Windows 上でテキストを選択してホットキーを押すだけで、日本語音声で読み上げてくれるシステムトレイ常駐アプリです。
ブラウザ、メモ帳、Word、PDF ビューアなど、コピーできるテキストならどこでも使えます。

---

## まず使ってみる（右クリックメニューから起動する場合）

セットアップ済みの方向けのクイックガイドです。

### 起動方法

1. **デスクトップの空いている場所でマウスの右ボタンを押す**
2. メニューから **「読み上げアプリを起動」** をクリック
   - 表示されない場合は **「その他のオプションを表示」** の中を確認してください（Windows 11）
3. タスクトレイ（画面右下）に **「読」** アイコンが出れば起動完了

> 右クリックメニューへの登録はレジストリに保存されるため、**電源を切っても登録は消えません**。
> 次回 Windows 起動後も同じ手順で呼び出せます。

### 使い方（起動後）

| 操作 | 動作 |
| ---- | ---- |
| テキストを選択 → **Ctrl + Alt + R** | 選択した文章を読み上げ |
| 任意の場所をクリック → **Ctrl + Alt + E** | クリックした位置から文末まで読み上げ |
| テキストを選択 → **Ctrl + Alt + S** | 選択した文章を生成AIで**要約**してから読み上げ |
| **Tab** | 現在の文節が終わったところで一時停止（もう一度で再開） |
| **Shift + Tab** | オーバーレイの位置を下 ↔ 上 で切替（読み上げ中のみ） |
| **Esc** | 読み上げを即座に中止 |

詳しいインストール手順・設定方法は下記をお読みください。

## 主な機能

| ホットキー | 動作 |
| ---------- | ---- |
| **Ctrl + Alt + R** | 選択中のテキストを読み上げ（もう一度押すと停止） |
| **Ctrl + Alt + E** | カーソル位置から文末までを読み上げ |
| **Ctrl + Alt + S** | 選択テキストを生成AIで要約してから読み上げ |
| **Tab**            | 読み上げ中に現在の文節が終わったところで一時停止 / 再開 |
| **Shift + Tab**    | 読み上げ中にオーバーレイ位置を画面下 ↔ 上で切替 |
| **Esc**            | 読み上げ中に押すと即座に停止 |

- 画面下部に「読み上げ中の文」を常時最前面で表示するオーバーレイ付き（Web ページが勝手にスクロールしても今どこを読んでいるか一目で分かります）
- MP3 生成中は柔らかいビープ音で動作中であることを通知
- 長文でも文節単位にチャンク分割し、先頭から順次再生（最初の一文が生成できた時点で読み始めます）
- 次の文節を並行して先読み生成することで、チャンク間の無音ギャップなく自然につながる
- タスクトレイアイコン「読」から停止・終了が可能

## 動作環境

- Windows 10 / 11 (64bit)
- Python 3.10 以上（3.13 で動作確認）
- インターネット接続（Microsoft edge-tts による音声合成に必須）

## インストール手順（初めての方向け）

### 手順 1. Python をインストールする

1. [https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/) から Python 3.10 以上のインストーラをダウンロード
2. インストーラ起動時、**「Add python.exe to PATH」 にチェックを入れて** 「Install Now」 をクリック
3. インストール後、コマンドプロンプトを開いて確認:

   ```cmd
   python --version
   ```

   `Python 3.13.x` のように表示されれば OK。

### 手順 2. 本リポジトリを取得する

Git がある場合:

```cmd
git clone https://github.com/araiai9801/yomiage.git
cd yomiage
```

Git が無い場合は GitHub の `Code → Download ZIP` で ZIP をダウンロードし、任意の場所に展開してください（例: `C:\Users\<あなた>\yomiage`）。

### 手順 3. 必要ライブラリをインストール

コマンドプロンプト（または PowerShell）でフォルダに入り、以下を実行:

```cmd
pip install pystray pyperclip pillow edge-tts
```

tkinter は Python 標準付属なので追加インストール不要です。

### 手順 4. 起動する

`yomiage.bat` をダブルクリック。
タスクトレイ（画面右下の時計の近く）に **「読」** というアイコンが出れば起動成功です。

> `pythonw.exe` と `yomiage.py` のパスはスクリプトが自動検出します。手動でのパス書き換えは不要です。

## 使い方

1. ブラウザでも Word でも、読み上げたいテキストをマウスで選択
2. **Ctrl + Alt + R** を押す
3. 画面下部に「読み上げ中」オーバーレイが出て、日本語音声で読み上げが始まります
4. 一時停止したいときは **Tab**（現在の文節が終わった後で止まります）
5. 完全に止めたいときは **Esc** か **Ctrl + Alt + R** をもう一度

**カーソル位置から最後まで読む**（長い Web 記事を途中から聴きたいとき便利）:
テキスト中の任意の場所をクリックしてカーソルを置き、**Ctrl + Alt + E** を押します。

**終了するには:**
タスクトレイの「読」アイコンを右クリック → 「終了」

## （任意）便利設定

### A. Windows 起動時に自動起動

PowerShell を **管理者として実行** し、本フォルダで:

```powershell
.\setup_startup.ps1
```

ログオン 30 秒後に自動起動するタスクが登録されます。
解除したいときはタスクスケジューラで `yomiage` タスクを削除してください。

### B. デスクトップ右クリックメニューから起動

PowerShell を **管理者として実行** し:

```powershell
.\register_context_menu.ps1
```

デスクトップや任意のフォルダの空白を右クリック → 「読み上げアプリを起動」 が追加されます。
解除は `.\unregister_context_menu.ps1` を実行。

## トラブルシューティング

| 症状 | 原因・対処 |
| ---- | ---------- |
| トレイアイコンが出ない | コマンドプロンプトで `python yomiage.py` と実行してエラーを確認。多くは `pip install` 不足 |
| `ModuleNotFoundError: pystray` など | 手順 3 の `pip install` をやり直す |
| 音が出ない | Windows の音量・既定の再生デバイスを確認。機内モード時は edge-tts が失敗します |
| 「音声を取得できませんでした」と表示 | ネット接続 / プロキシ / ファイアウォール設定を確認 |
| Ctrl+Alt+R が効かない | 他のアプリ（OBS など）が同じキーを登録している可能性。競合アプリを終了 |
| Ctrl+Alt+E を押しても読まれない | 対象アプリが Ctrl+Shift+End 選択に対応していないケース。ブラウザ・メモ帳・Word では動作します |
| 読み上げが途中で止まる | 長文は文単位で分割して再生。ネット断などで途切れた場合は Esc → 選び直し |

ログは `yomiage.log` に出力されます。問題報告の際に添付してください。

## 要約読み上げ機能（Ctrl + Alt + S）

長い文章を生成AIで要約してから読み上げる機能です。
ニュース記事や長い社内文書を、ざっくり把握したいときに便利です。

### 必要なもの

1. **OpenAI / Anthropic / Gemini いずれかの API キー**
2. 環境変数にAPIキーを設定（下記参照）
3. `config.toml`（初回起動で自動生成、必要なら編集）

### API キーを環境変数に設定する

#### 方法 A: GUI（おすすめ）

1. Windows キーを押し「**環境変数**」と入力
2. 「システム環境変数の編集」 → 「環境変数(N)...」ボタン
3. 上半分「ユーザー環境変数」の「新規(N)...」
4. 変数名にいずれかを入力：
   - OpenAI を使うなら → `OPENAI_API_KEY`
   - Anthropic Claude を使うなら → `ANTHROPIC_API_KEY`
   - Google Gemini を使うなら → `GEMINI_API_KEY`
5. 変数値に取得したAPIキーを貼り付け
6. OK で全て閉じる
7. **yomiage を再起動**（既に起動していたら一度終了させる）

#### 方法 B: PowerShell

```powershell
[Environment]::SetEnvironmentVariable("OPENAI_API_KEY", "sk-...", "User")
```

### config.toml で動作を変更

初回起動時に `config.toml` がスクリプトと同じフォルダに自動生成されます。
テキストエディタで開いて編集 → yomiage を再起動 で反映されます。

主な項目：

```toml
[summary]
provider = "openai"           # "openai" / "anthropic" / "gemini"
model = "gpt-4o-mini"         # 各プロバイダのモデル名
length = "medium"             # "short" / "medium" / "long" / "bullet"
extra_instruction = "話し言葉で簡潔に"
on_error = "fallback"         # API失敗時の動作: "fallback" / "skip"
timeout_s = 30                # API タイムアウト秒数
```

### 使い方

1. 長い文章をマウスで選択
2. **Ctrl + Alt + S** を押す
3. オーバーレイに「🤖 要約中...」と表示（数秒）
4. 要約が完成したら通常の読み上げと同じように再生

API失敗時（ネット切断・APIキー未設定など）は `on_error = "fallback"` の場合、
元のテキストをそのまま読み上げます。

### コスト目安（OpenAI gpt-4o-mini）

- 1万字の要約 = 約 0.02 円
- 月数百回使っても **数十円程度**

## アンインストール

1. 読み上げアプリを終了（トレイ → 終了）
2. タスクスケジューラ登録済みなら `yomiage` タスクを削除
3. 右クリックメニュー登録済みなら `unregister_context_menu.ps1` を実行
4. フォルダごと削除

## 仕組み（開発者向けメモ）

1. `RegisterHotKey` Win32 API でグローバルホットキー（Ctrl+Alt+R / Ctrl+Alt+E）を登録
2. ホットキー検出時、`SendInput` で Ctrl+C（または Ctrl+Shift+End → Ctrl+C）を送信し、選択テキストをクリップボードへ
3. `edge-tts` で Microsoft Nanami Neural 音声の MP3 を文節単位で生成（最大 50 文字/チャンク）
4. 現在のチャンクを再生しながら次のチャンクを並行して先読み生成（シームレス連続再生）
5. `winmm.dll` の MCI API で MP3 を直接再生（PowerShell 不要）
6. 再生中は Tab（チャンク境界一時停止）・Esc（即座停止）を動的に登録
7. tkinter で常時最前面のオーバーレイを別スレッド表示
